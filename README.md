# Проект: Место

**Описание проекта**

Mesto - сервис для размещения фотографий.
Авторизованные пользователи сервиса могут добавлять и удалять фотографии, а так же ставить лайки понравившимся фотографиям.


**Применяемые технологии**

- HTML5
- CSS3
- Java Script
- React JS
